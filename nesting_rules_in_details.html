Elements' rules

  • All tags must be closed.
  • When you nest an element inside of other one you must be careful. If open an element A and B is nested inside A, first, you must close the element B then you can close the element A.
  • All tags should be nested in proper order

Form.

  • All tags must be closed.
  • Attribute value must appear within quotes.
  • No form element inside a form element.
  • Nest all the content inside of the form element.
  • The form element must have a submit button.

List element.

  • All tags must be closed.
  • An unordered list <ul> is a list in which the order of the list items does not matter.
  • Ensure that all list item <li> elements are wrapped inside of <ul> or <ol> parent elements.
  • For the element to be valid, it must have both parent and child element. Parent element can be <ul> or <ol> and the child elements must be wrapped inside of the <il> tags.
  • You can not have paragraph inside of a list element
  • You must use ordered list <ol> if you are listing things that are need to be in order.

Bad nesting 

<main>

  <SECTION>

    <h2>

      hello world.
  </H2>

  <Ul>

    <li>

      good
  </Ul>

    <li>

      excellent
    <li>

</MAIN>

Good nesting. 

  <main>

    <section>

      <h2>

        Hello world.
      </h2>

      <ul>

        <li>

          good.
        </li>

        <li>

          excellent
        </li>

      </ul>

    </section>

  </main>
