Fundamental attribute rules.

  • Attribute must have name value pairs.
  • Be clear about the attribute functions.
  • Class attribute is for public, define by users.
  • Don't double booleans attribute.
  • Id attribute is for identity, define by users
  • Most tags have their own attribute.
  • Height attribute has numerical value.
  • Name attribute must be lowercase.
    Do not do this <option value="HELLO">Hello
    Do this <option value="hello">Hello
  • No double quotes inside of double quotes.
    Do not do this <input type=""text"">
    Do this <input type="text">
  • No space in attribute names, instaed use underscores or dashes.
    Do not do this <input type="text" name="artist name">
    Do this <input type="text" name="artist_name">
  • width attribute has numerical value.
  • Remove empty attributes
  • Sort class names.
  • The attribute title is used rarely. It adds a title to every element's content.

Bad value: Don' t do this 

  <main>
    <section>
      <h2>
        hello world
      </h2>
      <option value=""DUCK"">Duck
      <input type="TEXT" name="USERNAME">
    </section>
  </main>

Good value: Do this 

  <main>
    <section>
      <h2>
        Hello world
      </h2>
      <option value="duck">Duck
      <input type="text" name="username">
    </section>
  </main>
