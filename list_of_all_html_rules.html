List of all HTML5 rules

There are many rules for coding or writing HTML.

  • All tags must be closed.
  • Always add the alt attribute to images.
  • Attributes are separated by spaces
    • Attribute values must appear within quotes.
  • Avoid long lines code.
  • Collapse white spaces.
  • Child element must be indented.
  • Do not add blank lines without any reason.
  • Elements can contain element
  • Ending tags must start with: < and end with: >
  • Every document must have at least: <title>, <head> and <body>.
  • Every document must start with <!DOCTYPE html>
  • Every element must have opening tags
  • Every element should have their content.
  • Html element can be nested
  • Start tags must start with: < and end with: >
  • Tag names cannot contain spaces.
  • The language in every document must be precised by <html lang="...">
  • Use lowercase for attribute names
  • We can' t have an end tag without a staring tag
  • Write html as simple as possible
  • Leave everything out that is not absolutely necessary.
  • Think through your markup.
